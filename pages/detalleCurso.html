Descripción del curso

Este curso te enseñará los fundamentos del desarrollo web, incluyendo HTML, CSS y JavaScript. Aprenderás a crear sitios web responsivos y atractivos.

Contenido del curso

HTML:

  • Estructura básica de un documento HTML
  • Etiquetas principales
  • Listas: ul - ol
  • Formularios y elemento
  • Etiquetas semántica

CSS:

  • Modelo de caja: padding, margin, border
  • Colores y fondos
  • Flexbox y sus propiedades
  • Posicionamiento (relative, absolute, fixed)
  • Media queries para diseño responsive

JavaScript:

  • Manipulación del DOM
  • Manejo de eventos (`click`, `mouseover`, etc.)
  • Funciones y flechas (`=>`)
  • Promesas y manejo de asincronía
  • Validación de formularios con JavaScript
